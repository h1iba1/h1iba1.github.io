# 1.计算公式

公钥（e,n)

私钥（d,n)

M明文

C密文



C=M^e mod n



M=C^d mod n



n=q*p



fie(n)=(q-1)*(p-1)



d=(k*fie(n)+1)/e  (k为任意，使d为整数的数）



# 2.实列演示

先构建公钥，n=p*q，所以先选取两个质数

p=51,q=53

n=2703

fie(n)=2600

e=3

d=(2600k+1)/3 (k=1时）

d=869



所以得到公钥（3，2703），私钥（869，2703）



# 3.手算

明文为：23

求密文：

C=m^e mod n

C=23^3 mod 2703



密文：36

求明文：

M=C^d mod n

M=36^869 mod 2703
