# 取反原理：

取反分按位取反（~）和取反（!)

按位取反“~”：按位取反1变0，0变1

取反：逻辑取反，flase变true,true变flase



所以：

## 按位取反：

```php
$a=~11111010;
echo "$a";          //-11111011
echo base_convert($a,2,10);//251
```

## 取反：

```php
$b=!5;
echo var_dump("$b");//0
```



## 采用取反来构造任意字符串：

```php
$str='phpinfo';
echo "$str".':'.urlencode(~$str);//phpinfo:%8F%97%8F%96%91%99%90
```
